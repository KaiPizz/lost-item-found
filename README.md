# Odnalezione Zguby (Lost Items Found)

> HackNation 2025 Project

A 5-step wizard application to publish lost items as machine-readable data for dane.gov.pl.

## Project Structure

```
/frontend  – React + TypeScript + Vite + TailwindCSS
/backend   – Node.js + Express + TypeScript
/spec      – Data schema files
```

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend will start at `http://localhost:5173`

### Backend

```bash
cd backend
npm install
npm run dev
```

The API will start at `http://localhost:3000`

## Development

Run both frontend and backend in separate terminals for full-stack development.
